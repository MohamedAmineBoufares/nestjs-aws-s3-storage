## Description

A repository for using AWS S3 file upload with NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

App will be running [on port 5000](http://localhost:5000/)
